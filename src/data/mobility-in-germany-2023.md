# Mobility in Germany 2023 (MiD - Mobilität in Deutschland)

## Overview
This dataset contains trip duration distributions by trip purpose and transport mode.
Data basis: Trips in Germany (Wege in Deutschland)
Data collection method: CATI (Computer Assisted Telephone Interview) and CAWI (Computer Assisted Web Interview)
Note: Includes regular work-related trips, imputed values. Only trips with purpose shopping, errands, or leisure.
